Am 3 foldere i - de la Interface, s - de la Server si c - de la Client
1. Copiez cate un MyAnt.bat in fiecare folder si il configurez
2. Deschid un cmd din i
3. Rulez MyAnt Compile
4. Deschid un cmd din s
5. Rulez MyAnt Compile
6. Rulez MyAnt Rmi -> pornesc serviciul RMI si las fereastra deoparte
7. Deschid alt cmd din serviciul
8. Rulez MyAnt Server -> pornesc serverul si il las deoparte; trebuie sa zica ceva cum ca e Ready
9. Deschid un cmd din c
10. Rulez MyAnt Compile
11. Rulez MyAnt Run
12. Inchid ferestrele in care ruleaza serverul si RMI-ul cu CTRL+C
